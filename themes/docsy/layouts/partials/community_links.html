Editing Pages

Any of the course pages is editable on the web via links on the right sidebar. This will show the editing interface in github. After editing, click commit changes button at the bottom. The changes will be reviewed and published soon. See the Formats section for some special syntax.

Adding & Translating Courses

For adding a new course or a new translation of an existing course, please file an issue in github. Some files, folders and configurations needs to be added. If your are familiar, feel free to do it yourself and sent a pull request.

Develop Code

The system is based on Docsy theme of the Hugo static site generator. You can send pull requests to: https://github.com/opencurriculum-in/opencurriculum-in.github.io_code.
